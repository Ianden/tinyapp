# tinyapp
tinyapp project for lightouse labs

## uses
* shorten urls
* pretend the year is 1998 and all websites look this basic

## dependencies
* node.js 
* express
* ejs
* bcrypt
* body-parser
* cookie-session

## get started

`npm install`
`node server.js`
